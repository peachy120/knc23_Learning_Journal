HTML

  • Hyper-Text Markup Language
  • Determines the structure of the Web page
  • Written for both human user and browers
  • Uses Tags : Open and Close
  • Attributes are placed in the opening tag, like: id, class

HTML Attributes

  • Placed in the opening tag, like: id, class
  • Take values, then convey more information about the element
  • Help with Styling and JavaScript

HTML Semantic

  • header : Defines the introductory section of a web page. Contains navigation, theme switcher and search bar
  • nav : Specifies the navigation item of the page
  • main : Contain main section
  • article : Represents a certain part of a web page that conveys some particular information
  • aside : Represents a sidebar on a website
  • footer : Accommodates items as quick links and copyright information

CSS

  • Cascading Style Sheets
  • Declarative Language
  • Can be embedded in HTML tags
  • Specifies how to content should be presented by a browser

JavaScript

  • An interpreted, weakly typed programming language with some unique quirks
  • Primarily created to add interactive behaviuor to web pages
  • Offical name is ECMA Script

Document Object Model

  • An Application Programming Interface (API) for HTML and XML document
  • Enable to Access, Add, Modify and Delete page elements
  • W3C standard
