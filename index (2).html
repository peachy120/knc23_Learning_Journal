There are rules from XHTML 1.0 Strict that needed to be followed

  • Elements MUST be correctly nested
  • Elements MUST always be closes
  • Elements MUST be written in lowercase
  • Attribute names MUST be in lowercase
  • Attribute value MUST be quoted

About Anchors

  • Page Markers that let user link to another part within the same web page or another webpage on the same site
  • Can be link to a list of contents at the top of a web page to section out content
  • Can also create a link from the bottom of each section to navigate back to the top of the same web page

CODE I learned during the Lab lesson :

For tutorial 1 :

HTML:

  • h1 , h2 and h3: use to make HEADINGs & SUB-HEADINGS

For tutorial 2 :

HTML:

  • time: in 'YYYY-MM-DD' form
  • li: list itmes within the list
  • ul: for UNORDERED list
  • ol: for ORDERED list
  • a: Anchors, can be given an 'id' attribute to identify them uniquely. Needs 2 steps to create an Anchor:
  • Create an Anchor at the HTML page you would like to link it to. For example : a id="XXX"
  • Create a text link to that anchor destination from another part of the page. For Example : a href="#XXX"
  • figure: use to markup an image that illustrates the web content
  • img: use to markup the source file of an image. It is a self-closing tag. Can have 'width' & 'height' attributes, should be included within the tag. Usually seen as img src="Filename/photoName.(png/jpg)" width="XX" height="XX" alt="description of image"
  • figcaption: use to make up a caption for the image or illustration. It is nested with in the figure tags
  • br: use after a caption, to make a line break between the caption and copyright info
  • ©: the HTML markup for copyright character entity

The W3C defines HTML and CSS as follows:

Selectors - target an element by matching the name of a selector with an element in HTML.

  • Type selector : matches the name of the selector with an HTML tag
  • p {} = p
  • ID selector : matches the name of the selector with the value of an id attribute
  • It is specific id attribute so it can only be used once only in the code
  • It must have a UNIQUE value
  • E.g. #wrapper {} = div id="wrapper"
  • Class selector : matches the name of the selector with the value of the class attribute
  • The same class attribute can be used more than once in the code.
  • E.g. .post {} = header class="post"
