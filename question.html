Question 2

A block element:

  1. continues on the same line as previous element
  2. takes up the full available width
  3. can only contain inline element

Answer

Question 3

  1. using media queries to keep the layout the same for all device sizes
  2. using flexible units like: fr and vw

Question 5

Write a function in scripts/code.js called makeFilename that replaces all space (" ") characters in a string with underscore ("_") characters. The function should take a single string parameter and return the string resulting from the operation.

Answer

function makeFilename(s) {
return s.split(" ").join("_");
}
