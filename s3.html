Session 3 Tutorial Questions

2. Print to the console first the expression (1 == true) and then the expression (1 === true).

Note the difference between using the simple and strict comparison operators == and ===.

Answer :

console.log("(1 == true):", 1 == true);
console.log("(1 === true):", 1 === true);

3. Define two variables x and y and initialise them to 3 and 5.

Answer :

let x = 3;
let y = 5;

a. Print to the console the following expressions:

i. x less or equal to y

Answer :

console.log("x <= y:", x <= y);

ii. x greater than 4 and y greater than 4

Answer :

console.log("x > 4 && y > 4:", x > 4 && y > 4);

iii. x greater than 4 or y greater than 4

Answer :

console.log("x > 4 || y > 4:", x > 4 || y > 4);

iv. not (x greater than 4 or y greater than 4)

Answer :

console.log("!(x > 4 || y > 4):", !(x > 4 || y > 4));

v. x not equal y and y greater than 0

Answer :

console.log("x !== y && y > 0:", x !== y && y > 0);

vi. x times y less than 15

Answer :

console.log("x * y < 15:", x * y < 15);

vii. x times y equal or less than 15

Answer :

console.log("x * y <= 15:", x * y <= 15);

b. Use an if statement that tests whether x is greater than y and then prints either "x is greater than y" or "x is less than or equal to y" to the console.

Answer :

if ( x > y ) {
    console.log("x is greater than y")
} else {
    console.log("x is less than or equal to y")
}

c. Log the result of dividing x by y only if x is greater than y

Answer :

if ( x > y) {
    let result = x / y;
    console.log(result)
}

d. Add x to y while y is less than 100, then log the value of y to the console

HINT: use a while loop

Answer :

while ( y < 100) {
    y = y + x;
}

4. Log the string "Hello " to the console y times (i.e. Hello 1, Hello 2, Hello 3 ....)

HINT: use a for loop

Answer :

for ( let counter = 0 ; counter < y ; counter++ ) {
    console.log("Hello" + counter);
}

5.Define a variable day and initialise it to a random number between 0 and 6. Then use a switch statement to test the value of day and print a corresponding day of the week to the console. (0=Monday, 1=Tuesday, ..., 6=Sunday)

Hint: use the following expression to generate a random number between 0 and 6: var day = Math.floor(Math.random() * 7);

Answer :

var day = Math.floor( Math.random() * 7 );
switch(day) {
    case 0:
        console.log("Monday");
        break;
    case 1:
        console.log("Tuesday");
        break;
    case 2:
        console.log("Wednesday");
        break;
    case 3:
        console.log("Thursday");
        break;
    case 4:
        console.log("Friday");
        break;
    case 5:
        console.log("Saturday");
        break;
    case 6:
        console.log("Sunday");
        break;
    default:
        console.log("Invalid Day")
}
